• Prefix: AT
  • Suffix: \r
AT+로 시작해도 되는데 그건 명령 여러 개 쓸 때 사용함
명령 1개 써도 +써도 됨
세미콜론으로 명령 구분

AT+CMD1;CMD2\r

Commands

UART_DEF UART 통신 설정 <Baud rate>, <Data bits>, <Stop bits>, <Parity bits>, <Flow control>
CIPSTATUS 연결된 클라이언트 정보 조회 (아이디, 아이피 주소 등)
CIPSEND 클라이언트로 송신 <Client ID>, <Length> 이후 Length만큼 데이터 입력
CIFSR 아이피 조회
CWJAP? 연결된 와아파이 조회
