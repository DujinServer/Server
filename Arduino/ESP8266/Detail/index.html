1. 보드레이트 설정

esp 기본 보드레이트가 115200이라서
9600 으로 바꿔줘야 함

아두이노랑 esp 둘 다 115200으로 설정하면
가끔 글자 깨져서 9600으로 맞춤
esp 115200으로 시작 
esp.begin(115200);
AT 명령으로 보드레이트 교체 
AT+UART_DEF=9600,8,1,0,0
esp 9600으로 교체 
esp.begin(9600);

2. 모드 설정

  1. Station
  2. AP
  3. Station + AP
스테이션 모드 쓸거니까 1로 설정
AP는 공유기처럼 쓰는 거 같은데 정확히 모름
3번 둘 다 사용으로 해도 되긴 한데 굳이 그럴 필요 없음

AT+MODE=1

3. 와이파이 연결

Syntax 
AT+CWJAP="SSID","PW"
Example 
AT+CWJAP="iptime mini","1234"

비밀번호 없으면 SSID만 입력

Setup 함수

esp 꺼졌다가 다시 켜질 때 서버 다시 켜줘야 함

AT+CIPMUX=1

MUX는 여러 클라이언트 동시에 연결
	0: 단일
	1: 멀티

클라이언트 연결 순서대로 0부터 아이디 부여
아이디 0~4까지 최대 5개까지 연결 가능

서버 시작려면 MUX 1로 설정해야 함
MUX 0으로 설정할 때도 서버 열려있으면 설정 안됨
서버 먼저 끄고 MUX 0으로 교체해야 함
Syntax 
AT+CIPSERVER=MODE,PORT

Mode
	0: Off
	1: On

Port: 서버 포트 (0은 안됨, 1부터 가능)
Example 
AT+CIPSERVER=1,1
